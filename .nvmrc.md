# Node.js Version Management

This project uses Node.js v22.x as specified in `.nvmrc` and `package.json`.

## Quick Start

If you have `nvm` installed:

```bash
# Automatically use the correct Node.js version
nvm use

# Or manually
nvm install 22
nvm use 22
```

## Why Node.js v22?

- Node.js v25 removed the deprecated `http_parser` module
- The `restify` package (v11.1.0) still depends on `http_parser`
- Node.js v22 is the latest LTS that supports `http_parser`

## Verify Your Version

```bash
node --version
# Should output: v22.x.x
```

If you see `v25.x.x` or higher, you need to switch to v22.
